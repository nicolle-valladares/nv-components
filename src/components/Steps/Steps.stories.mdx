# Steps

`import { Steps } from '@nicolle-valladares/ra-components';`

`Steps` is a _navigation_ bar that guides users through the steps of a task. Incidentally, it can be used to display the progress of a task

## When To Use

When a given task is complicated or has a certain sequence in the series of subtasks, we can decompose it into several steps to make things easier.
